# 天气预报使用说明

## 数据源说明

本项目使用**Open-Meteo**作为数据源，这是一个完全免费、无需API密钥的天气API服务。

## 数据源特点

- ✅ 完全免费，无需注册
- ✅ 无需API密钥
- ✅ 支持全球所有城市
- ✅ 数据准确可靠
- ✅ 无请求次数限制

## 功能特性

- ✅ 支持全球所有城市
- ✅ 实时天气数据
- ✅ 7天天气预报
- ✅ 简洁大气的界面
- ✅ 高级动态效果
- ✅ 响应式设计

## 启动服务

在项目目录下运行：

```bash
.\start-server.bat
```

然后在浏览器中打开：http://localhost:8000/weather.html

## 使用方法

1. 在搜索框中输入城市名称（中文或英文）
2. 点击搜索按钮或按回车键
3. 查看实时天气和7天预报

## 支持的城市

支持全球所有城市，包括：
- 中国所有城市（北京、上海、广州等）
- 国际城市（东京、纽约、伦敦、巴黎等）

## 技术栈

- HTML5
- CSS3（动画效果）
- 原生JavaScript（无需框架）
- Open-Meteo API（天气数据）

## 注意事项

- 数据来源：Open-Meteo（基于欧洲中期天气预报中心）
- 数据更新：实时更新
- 网络要求：需要联网访问API
